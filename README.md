# 🔐 Advanced Digital Forensics & Steganography Suite

A professional cybersecurity and digital forensics toolkit for detecting hidden payloads, steganography, embedded files, metadata anomalies, and suspicious artifacts across images and documents.

---

## 🚀 Features

- 🔍 Advanced Steganography Detection
- 📁 Folder & Recursive File Scanning
- 🖼️ Image Analysis (PNG/JPG)
- 📄 PDF & DOCX Analysis
- 📦 Embedded File Carving
- 📊 Entropy & Statistical Analysis
- 🔐 LSB Steganography Detection
- 📑 Automated PDF Forensic Reports
- 🧠 Metadata & EXIF Inspection
- ⚠ Risk Classification Engine
- 🖥️ Professional Dark-Themed GUI

---

## ⚡ Installation

```bash
git clone https://github.com/YOUR_USERNAME/stego-forensics-suite.git

cd stego-forensics-suite

python -m venv .venv

# Windows
.venv\Scripts\activate

pip install -r requirements.txt
```

```bash
python forensic_gui.py
```
